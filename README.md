# Agora Bot
####  Agora is a Telegram group managment bot made using Telethon and Pyrogram which makes it modern and faster than most of the exisitng Telegram Chat Managers.

- Modern
- Fast
- Fully open-source
- Frequently updated
- Multi Language Support [Soon]

<details>
<summary><b>🔗 Deploy to VPS</b></summary>
<br>


```console
$ git clone https://github.com/Prince-Botz/TianaBot
$ cd TianaBot
$ pip3 install -U -r requirements.txt
$ cp sample.env .env
```
> Edit .env with your values and then start bot with
```console
$ bash start
```
